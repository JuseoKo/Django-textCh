번역할 문장
번역된 문장

1. 과제 추진배경

코드와 스크립트(텍스트)가 섞여있는 게임과 같은 문서를 번역할 때 버전업이 되면서
기존 번역문서를 날리고
처음부터 다시 번역해야 하는 문제가 존재한다.

이 프로젝트를 이용하여
한번 번역된 문서는 다음에도 그대로 번역할 수 있게끔
저장 된 텍스트를 변환해 주는 프로그램을 개발하고자 한다.

2. 과제(주제) 목표 및 내용

한번 번역된 문장들을 서버에 등록해 놓고 소스파일을 서버에 전송하면 자동으로 등록된 문장만 번역해준다.

웹 프레임 워크로는 Django를 사용하고 기능적인 부분은 Python으로 구현한다.

웹은 Css와 javaScript 로
구현하고 데이터시트는 Json형식으로 구현한다.

시간이 남는다면 회원기능을 추가해서 각 유저별로 데이터를 관리할 수 있게 한다.

유저별로 데이터를 관리할때는 github와 같이
팀을 이루어 작업할 수 있게끔 그룹작업 기능을 구현한다.

5. 기대효과 및 활용방안

번역해야될 문장과
번역하지 않아야 할 문장이
섞여있는 텍스트 모음
(대표적으로게임)에서
다시 번역하지 않아도 된다.

이로부터 번역 작업을 하는 사용자의 노동과 시간 단축을 기대할 수 있다.

자금 사정이 열악한
인디게임 제작에 협력하여
각종 언어로 쉽게
번역할 수 있게 지원하여
초기 개발비용을 줄여주는
효과를 낼 수 있다.

또한 외국어로 이루어진 게임의 경우 철저히 개인번역에만 기대야 하는데
이를 팀을 이루어 번역하기 쉽게끔 서비스를 제공한다.
